EDUSCHOLAR (Student Marks and Attendance Management System)

Softwares used:

Front End - HTML, CSS, JavaScript, BootStrap

Back End - PHP (8.0.1)

Distribution - XAMPP (3.2.4)

Database - MySQL (MariaDB 10.4.17)

Code Editor - VSCode (1.63.2)

To run the project:

1) Install Visual Studio Code:
https://code.visualstudio.com/download

2) Download Xampp:
https://www.apachefriends.org/download.html 

3) Extract the source code and put it in htdocs folder of Xampp

4) Open Xampp Control Panel and Start Apache server and MySQL

5) Open PHPMyAdmin in the browser and import the database file

6) Run the local host server in any browser and view the website.
(Ensure proper internet connection.)
